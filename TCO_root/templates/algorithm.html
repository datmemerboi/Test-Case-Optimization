What is Meta-Heuristic Algorithm?

Meta-Heuristic Algorithms are high level procedural algorithms which are designed mainly to find, generate and select heuristics which would provide the best possible solutions for a given optimization problem. Meta-Heuristic algorithms generate a completely random set of solutions on an extremely large space which would then be optimized to find the best possible solution for the given problem. The advantage of meta-heuristic algorithms is there is no bias over the generated set of solutions. The solutions are completely random and, by each iteration, approach towards a universally optimal result across different solutions.

PSO

Particle Swarm Algorithm is a meta-heuristic algorithm which is manily used to optimize a problem by repeatedly trying to provide a solution with regard to measure of general quality among the candidate solutions.

GA

Genetic Algorithm is a meta-heuristic algorithm inspired from the working of natural selection, belonging to a large list of evolutionary algorithms. It is commonly used to generate solutions of high quality for problems of optimization.
