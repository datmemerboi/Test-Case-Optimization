TEST CASE OPTIMIZATION USING META-HEURISTIC ALGORITHMS

Project by Shravan Dave

A system to optimize test cases involved in the Software Testing using different Meta-heuristic Algorithms. This enables the tester to sort out and focus on the most important test cases.

For optimization, we use a Meta-Heuristic Algorithm (Particle Swarm Optimizer) and perform a comparative study with a different Meta-Heuristic Algorithm (Genetic Algorithm).
